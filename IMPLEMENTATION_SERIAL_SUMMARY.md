# Реализация системы серийных номеров продукции

## ✅ Выполненные задачи

### 1. Структура базы данных
Создана SQL миграция `/workspace/polesie/sql/migrations/001_serial_numbers.sql`:

- **product_serial_numbers** - таблица для хранения серийных номеров с:
  - Уникальным серийным номером
  - Индивидуальными техническими характеристиками (JSON)
  - Данными паспорта
  - Гарантийной информацией
  - Статусами (active, warranty, expired, returned, scrapped)

- **product_passport_versions** - история изменений паспортов
- **product_documents** - прикреплённые документы (руководства, сертификаты)

### 2. Веб-интерфейс

#### Страница управления серийными номерами
`/workspace/polesie/modules/production/serial_numbers.php`
- Список всех серийных номеров с фильтрацией
- Создание/редактирование/удаление
- Поиск по номеру и продукту
- Фильтры по статусу и продукту

#### Страница просмотра паспорта
`/workspace/polesie/modules/production/view_passport.php`
- Просмотр полной информации о продукте
- Отображение технических характеристик
- Список прикреплённых документов
- Версия для печати (A4 формат)
- Редактирование паспорта

#### API для работы с данными
`/workspace/polesie/modules/production/api_passport.php`
- Получение данных о серийном номере
- Обновление паспорта (с сохранением версий)
- Загрузка документов
- Автоматическая генерация серийных номеров

### 3. Интеграция
- Добавлен пункт меню "Серийные номера" в боковую панель
- Создана директория для загрузки документов

## 📋 Как использовать

### Шаг 1: Применение миграции
```bash
mysql -u username -p database_name < /workspace/polesie/sql/migrations/001_serial_numbers.sql
```

### Шаг 2: Доступ к системе
1. Откройте веб-интерфейс
2. В меню выберите **Производство → Серийные номера**
3. Нажмите **+ Добавить** для создания нового серийного номера

### Шаг 3: Заполнение данных
- **Серийный номер**: уникальный идентификатор (например, `SN-AIR-080-2-20250123-AB12`)
- **Продукция**: выбор из справочника
- **Технические характеристики**: JSON формат
  ```json
  {
    "power": "3.0 кВт",
    "voltage": "380В",
    "rpm": "3000 об/мин"
  }
  ```
- **Гарантия**: даты начала и окончания
- **Документы**: загрузка руководств и сертификатов

### Шаг 4: Печать паспорта
1. Откройте страницу паспорта (кнопка **📄 Паспорт**)
2. Нажмите **🖨️ Печать** для печати версии A4

## 🔑 Ключевые возможности

1. ✅ **Индивидуальный серийный номер** для каждого изделия
2. ✅ **Технические характеристики** хранятся в JSON (можно редактировать)
3. ✅ **Паспорт продукта** с возможностью печати
4. ✅ **Руководство по эксплуатации** и другие документы прикрепляются к изделию
5. ✅ **История изменений** паспортов (версионирование)
6. ✅ **Поиск и фильтрация** по всем параметрам
7. ✅ **Статусы** для отслеживания жизненного цикла
8. ✅ **Гарантийная информация**

## 📞 Интеграция с производством

Для автоматического создания серийных номеров при завершении производства:

```php
// Пример в обработчике завершения production_orders
$pdo->prepare("
    INSERT INTO product_serial_numbers 
    (serial_number, product_id, production_order_id, manufacture_date, status)
    VALUES (?, ?, ?, CURDATE(), 'active')
")->execute([$serialNumber, $productId, $orderId]);
```
