# 📄 Паспорта продуктов - Руководство пользователя

## Обзор

Система паспортов продуктов ОАО «Полесьеэлектромаш» содержит полную информацию о производстве всех 72 продуктов компании.

## Как открыть паспорта

### Через меню навигации
1. Откройте боковую панель (sidebar)
2. Перейдите в раздел **Производство**
3. Нажмите на пункт **📄 Паспорта продуктов**

Или напрямую перейдите по адресу: `/polesie/modules/products/passports.php`

## Функционал

### Главная страница паспортов
- **Карточки продуктов** - превью каждого продукта с основной информацией
- **Поиск** - по SKU или названию продукта
- **Фильтр по категориям** - 7 категорий продуктов:
  - Электродвигатели асинхронные трехфазные (36 продуктов)
  - Электродвигатели однофазные (12 продуктов)
  - Насосы (7 продуктов)
  - Литье чугунное (7 продуктов)
  - Электроплиты (6 продуктов)
  - Алюминиевое литье (2 продукта)
  - Custom продукты (2 продукта)

### Детальная информация о продукте
При клике на карточку продукта открывается модальное окно с полной информацией:

#### 📊 Основная информация
- Артикул (SKU)
- Категория
- Общий вес
- Гарантия

#### ⚙️ Технические характеристики
- Мощность, обороты, напряжение
- Высота оси, класс защиты
- Климатическое исполнение
- Варианты монтажа

#### 📦 Материалы для производства
Материалы сгруппированы по категориям:
- 🔩 Металлы
- 🔧 Комплектующие
- 🔩 Крепеж
- 🎨 Лаки и краски
- 🛡️ Изоляция
- 📦 Упаковка
- ⚡ Электрика

Для каждого материала указано:
- Название
- Код материала
- Количество
- Единица измерения

```json
      "quality_requirements": [...]

```

## Обновление паспортов

Для обновления или перегенерации паспортов выполните:
```bash
python3 generate_passports.py
```
