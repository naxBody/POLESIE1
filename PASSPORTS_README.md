# Паспорта продуктов - Руководство

## 📋 Обзор

Система паспортов продуктов для ОАО «Полесьеэлектромаш» содержит полную информацию о материалах, необходимых для производства каждого продукта.

## 📁 Файлы

| Файл | Описание |
|------|----------|
| `/workspace/passports.json` | Основной файл с паспортами всех 72 продуктов |
| `/workspace/generate_passports.py` | Скрипт генерации паспортов |
| `/workspace/polesie/modules/products/passports.php` | Веб-интерфейс просмотра паспортов |

## 🔧 Генерация паспортов

Для обновления или пересоздания паспортов:

```bash
cd /workspace
python3 generate_passports.py
```

## 📊 Структура паспорта

Каждый паспорт продукта содержит:

### 1. Основная информация (`basic_info`)
- `name_full` - Полное название продукта
- `name_short` - Краткое название
- `code_gost` - Код по ГОСТ
- `category` - Категория продукта
- `category_code` - Код категории

### 2. Спецификации (`specifications`)
Технические характеристики продукта:
- Мощность (кВт)
- Обороты (об/мин)
- Высота оси вращения (мм)
- Напряжение (В)
- И другие параметры

### 3. Материалы (`materials`)
Список материалов с количеством для производства:

| Материал | Код | Количество | Ед. изм. |
|----------|-----|------------|----------|
| Чугун серый СЧ15 | CAST-SC-15 | 1.44 | кг |
| Провод медный ПЭТВ | WIRE-CU-PETV-2.0 | 0.09 | кг |
| Сталь электротехническая | STEEL-ELECTR-2212 | 0.69 | кг |
| Подшипники 6200 серии | BRG-6200-series | 2 | шт |
| Крепеж | FASTENER-MIX | 0.64 | кг |
| Лак электроизоляционный | VAR-PES-55 | 0.34 | л |
| Изоляционные материалы | INS-COMBO | 0.22 | кг |
| Сталь конструкционная 45 | ST-BAR-45-STD | 1.06 | кг |

### 4. Производственные примечания (`production_notes`)
- Контроль балансировки ротора
- Пропитка обмоток лаком с сушкой
- Проверка сопротивления изоляции

### 5. Требования к качеству (`quality_requirements`)
- ГОСТ Р 51689-2000
- Класс нагревостойкости изоляции F (155°C)
- Степень защиты IP54/IP55

## ⚙️ Настройка коэффициентов расхода

В файле `generate_passports.py` можно настроить коэффициенты расчета материалов:

```python
# Для станины (чугун)
cast_iron_coeff = 0.8 + (shaft_height / 100) * 0.3

# Для меди в обмотках
copper_coeff = 0.05 + (power_kw / 10) * 0.08

# Для стали электротехнической
steel_electrical_coeff = 0.6 + (power_kw / 5) * 0.15
```

> ⚠️ **Важно**: Коэффициенты являются примерными и должны быть уточнены у технологов предприятия.
