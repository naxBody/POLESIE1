# Реализация: Просмотр документов ГОСТ

## Что было сделано

### 1. Изменения в файле `polesie/modules/warehouse/materials.php`

#### Добавлен функционал кликабельных ссылок на ГОСТы:

- **Функция `extractGostNumber()`** (строки 1852-1859):
  - Извлекает номер ГОСТ из строки стандарта (например, "ГОСТ 1050-2013" → "1050")
  
- **Функция `generateGostLink()`** (строки 1862-1871):
  - Генерирует ссылку на внешний ресурс docs.cntd.ru для просмотра документа
  - Формат ссылки: `https://docs.cntd.ru/document/{номер-гост}`

- **Обновлена функция `openMaterialModal()`** (строки 1617-1640):
  - При отображении спецификаций материала, поле "standard_doc" содержащее ГОСТ становится кликабельной ссылкой
  - Ссылка открывается в новой вкладке

## Как это работает

### Текущая реализация (онлайн):
1. Пользователь открывает карточку материала
2. В модальном окне в разделе "Характеристики" поле "Нормативный документ" отображается как ссылка
3. При клике на ссылку открывается страница с текстом ГОСТа на сайте docs.cntd.ru

### Альтернативная реализация (локальные файлы):

Если вы скачаете документы ГОСТ и сохраните их в папку `polesie/assets/gosts/`, измените функцию `generateGostLink()`:

```javascript
function generateGostLink(gostString) {
    const gostNumber = extractGostNumber(gostString);
    if (!gostNumber) {
        return '#';
    }
    
    // Для локальных файлов
    const yearMatch = gostString.match(/-(\d{4})$/);
    const year = yearMatch ? yearMatch[1] : '';
    const fileName = `GOST-${gostNumber.replace('.', '-')}${year ? '-' + year : ''}.pdf`;
    
    return `../../assets/gosts/${fileName}`;
}
```

## Формат именования файлов

При скачивании ГОСТов используйте следующий формат:
```
GOST-{номер}-{год}.pdf
```

Примеры:
- `GOST-1050-2013.pdf`
- `GOST-4543-2016.pdf`
- `GOST-14637-2016.pdf`

Для ГОСТов с точкой в номере (например, 21427.0) замените точку на дефис:
- `GOST-21427-0-2019.pdf`

## Проверка работы

1. Откройте страницу материалов: `http://your-domain/polesie/modules/warehouse/materials.php`
2. Кликните на любой материал
3. В модальном окне найдите поле "Нормативный документ"
4. Кликните на ссылку ГОСТ — должен открыться документ в новой вкладке
